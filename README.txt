	      How to send and decrypt an encrypted message:
	         1. Initial rotor setting
	         2. Rotor wiring if modified
	         3. Encrypted message

                 Example:
		 2 5 3
		 YRXXBSFTKPGWDUWCLRXBVHJJQ 

		 To decrypt a message, use the rotor up/down buttons to set the rotors to the configuration sent. Set wiring then save if needed.
		 Paste the encrypted message into the input box, and the decrypted message should be returned in the output box.

                 Note: To be accurate to the real machine, anything not in the alphabet will be returned as a Z, this includes numbers, spaces, and punctuation.
